# Лабораторная работа 7: Отсортированный связный список

## Предметная область: IT-компания

Проект демонстрирует модификацию шаблонного класса-коллекции `MyContainer<T>`. Вместо массива (исходная реализация из семинара 5) в качестве внутренней структуры хранения используется **отсортированный двусвязный список**.

---

## Выбранный вариант внутренней структуры

**Отсортированный двусвязный список (sorted doubly linked list)**

При добавлении элементов через `push_back()` они автоматически вставляются в отсортированном порядке. Контейнер всегда остаётся отсортированным без необходимости вызывать `sort()`.

### Почему выбран этот вариант

- Компромисс между сложностью реализации и функциональностью
- Наглядная демонстрация работы итераторов на связной структуре
- Хорошая база для понимания более сложных структур (деревья, хеш-таблицы)
- Сохранение внешнего интерфейса при полной замене внутренней реализации

---

## Структура узла

```cpp
struct Node {
    T data;      // данные
    Node* prev;  // указатель на предыдущий узел
    Node* next;  // указатель на следующий узел
};
```
